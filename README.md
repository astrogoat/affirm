An Affirm app for Stratum

## Installation

You can install the package via composer:

```bash
composer require astrogoat/affirm
```

## Usage

```php
$affirm = new Astrogoat\Affirm();
echo $affirm->echoPhrase('Hello, Astrogoat!');
```

## Testing

```bash
composer test
```
